# Constraint Adapter Contracts

**Phase 1 Output** | **Date**: 2026-03-06

## `@toride/prisma` — PrismaConstraintAdapter

### Package Export

```typescript
import { createPrismaAdapter } from "@toride/prisma";

// Factory function — creates an adapter for a specific Prisma model
const adapter = createPrismaAdapter({
  // Maps relation constraint fields to Prisma relation names
  // e.g., "projectId" → "project" (Prisma uses relation names, not FK columns)
  relationMapping?: Record<string, string>;

  // Maps has_role constraints to Prisma role assignment queries
  roleAssignmentTable?: string; // default: "roleAssignments"
  roleAssignmentFields?: {
    userId?: string;    // default: "userId"
    role?: string;      // default: "role"
  };
});
```

### Translation Rules

| Constraint Type | Prisma Output |
|----------------|---------------|
| field_eq | `{ [field]: value }` |
| field_neq | `{ [field]: { not: value } }` |
| field_gt | `{ [field]: { gt: value } }` |
| field_gte | `{ [field]: { gte: value } }` |
| field_lt | `{ [field]: { lt: value } }` |
| field_lte | `{ [field]: { lte: value } }` |
| field_in | `{ [field]: { in: values } }` |
| field_nin | `{ [field]: { notIn: values } }` |
| field_exists (true) | `{ [field]: { not: null } }` |
| field_exists (false) | `{ [field]: null }` |
| field_includes | `{ [field]: { has: value } }` |
| field_contains | `{ [field]: { contains: value } }` |
| relation | `{ [relationName]: childQuery }` |
| has_role | `{ [roleTable]: { some: { userId: actorId, role: role } } }` |
| unknown | `{}` (no-op, post-filter in app code) |
| and | `{ AND: queries }` |
| or | `{ OR: queries }` |
| not | `{ NOT: query }` |

## `@toride/drizzle` — DrizzleConstraintAdapter

### Package Export

```typescript
import { createDrizzleAdapter } from "@toride/drizzle";

// Factory function — creates an adapter for a specific Drizzle table
const adapter = createDrizzleAdapter(schema.tasks, {
  // Maps relation fields to Drizzle table references
  relations?: Record<string, { table: AnyTable; foreignKey: string }>;

  // Maps has_role to a role assignments table
  roleAssignments?: {
    table: AnyTable;
    userIdColumn: string;
    roleColumn: string;
  };
});
```

### Translation Rules

| Constraint Type | Drizzle Output |
|----------------|----------------|
| field_eq | `eq(table.field, value)` |
| field_neq | `ne(table.field, value)` |
| field_gt | `gt(table.field, value)` |
| field_gte | `gte(table.field, value)` |
| field_lt | `lt(table.field, value)` |
| field_lte | `lte(table.field, value)` |
| field_in | `inArray(table.field, values)` |
| field_nin | `notInArray(table.field, values)` |
| field_exists (true) | `isNotNull(table.field)` |
| field_exists (false) | `isNull(table.field)` |
| field_includes | `arrayContains(table.field, value)` |
| field_contains | `like(table.field, '%value%')` |
| relation | `exists(select from relatedTable where fk = table.id AND childQuery)` |
| has_role | `exists(select from roleAssignments where userId = actorId AND role = role)` |
| unknown | `sql\`true\`` (no-op) |
| and | `and(...queries)` |
| or | `or(...queries)` |
| not | `not(query)` |
